Web Compatibility Sprint

Web Compatibility, the ability for all Web content to run in all browsers, is important. Hay muchos motores web de contenido que se ejecutan en varios dispositivos y, a veces, con pequeñas pruebas y algún que otro cambio en la página, los desarrolladores pueden garantizarles a los usuarios una experiencia completa. Test sites and spread the message about compatibility. [Descubre más](/es/webcompat-sprint/).
